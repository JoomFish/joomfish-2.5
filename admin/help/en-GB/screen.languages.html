Languages

Introduction

Translations are based on languages. Within the Joomla! system all languages can be related to the static language files. These files include the translation of all static text within your system.

Joom!Fish uses these files to determin which languages are available for your installation. You can download and install new files on the Joomla! extensions directory.
After installation of these files you can configure the details of your language, such as the name, URL short code and also the site configuration settings for this language in this dialog.
